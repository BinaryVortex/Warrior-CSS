# Warrior CSS

![Warrior CSS Demo](Screenshot%202026-04-13%20130542.png)

A pure CSS artwork creating a cartoon warrior character using only HTML divs and CSS styling. No images, no JavaScript—just creative CSS techniques!

## Demo

Check out the live demo: [https://binaryvortex.github.io/Warrior-CSS/](https://binaryvortex.github.io/Warrior-CSS/)

## How it Works

The warrior is built using positioned divs with CSS classes that define shapes, colors, and effects:

- `.cartoon`: Main container
- `.head`, `.body`, `.leg`: Body parts
- `.shield`: Protective gear
- `.lance`: Weapon
- `.point`: Spear point
- Utility classes like `.b` (border), `.r` (round), `.hb` (has before), `.ha` (has after)

## Usage

1. Clone or download the repository
2. Open `index.html` in your browser to see the warrior
3. Customize the CSS in `style.css` to modify the appearance

To use in your own project, copy the relevant CSS classes and HTML structure.

## Features

- Pure CSS artwork
- Responsive design using vmin units
- Creative use of box-shadow, gradients, and clip-path
- No external dependencies

## Browser Support

Works in modern browsers that support CSS features like clip-path, gradients, and box-shadow.
